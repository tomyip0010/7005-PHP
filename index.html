Web-Dev Software Development Environment

Welcome to Web-Dev Software Development Environment for David Chen's web development courses. This server contains tools for developing web application using Node.js, Apache 2.4, Composer (to install Laravel), and mysql-client.

  • The apache web server at the this URL...

    https://{user}.elf.ict.griffith.edu.au/

    refers to the "~/html/" directory of your environment's working home. Hence the html file you are reading now resides in the file "html/index.html".

    You are free to create other files and directories under "~/html/" to house your web projects and exercises (in HTML, PHP or Laravel).

  • Your "Node.js" application should be created outside the Apache "html" sub-directory, and can use ports 8080 or 8888 (HTTP protocol)... For example if using 8888, then when run it will be accessible on the URL...

    https://{user}.elf.ict.griffith.edu.au:8888/

    If you setup "Node.js" to use HTTPS protocol directly, then you can use ports 3000, or 3001, to make it accessible.

  • To install the Laravel Software, run

    composer global require laravel/installer

    To create a laravel server project, please read Creating a secure laravel project. This is important information as it describes how to create a sub-directory containing a secure laravel project. Do not rely on other similar information that can be found on the web.

  • For more information about this server and the software development environments you are using, see the servers top-level pages at

    https://elf.ict.griffith.edu.au/
